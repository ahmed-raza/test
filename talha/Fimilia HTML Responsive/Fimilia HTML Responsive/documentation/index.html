Edit Icons

There is a folder "icons", with png files. Find your required icon from folder and copy its name, and put name as class of your element but after icon- , like :

                    class="icon-iconname"

Headers Changes

There is a sticky header, you may change affix position there is offset position 400px you may change its affix size.

                    data-spy="affix" data-offset-top="400"

Home page 5 have different classes in slider and feature sections

                    "slider5" for slider section.
                    "features5" for featured section under the slider.
